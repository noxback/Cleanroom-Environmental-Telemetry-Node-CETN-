# Cleanroom-Environmental-Telemetry-Node-CETN-

## Overview
The Cleanroom Environmental Telemetry Node (CETN) is an Industrial Internet of Things (IIoT) monitoring device designed to collect, display, and eventually transmit critical environmental data commonly monitored in semiconductor, pharmaceutical, and advanced manufacturing cleanrooms.

The long-term objective of this project is to design a professional-quality embedded monitoring system that demonstrates hardware design, PCB development, embedded software, sensor integration, industrial communications, and engineering documentation.

This project is being developed as a personal engineering portfolio and learning platform while applying concepts from Electrical Engineering, Physics, embedded systems, and industrial automation.

---

## Project Goals

- Develop an embedded environmental monitoring system
- Design custom PCB hardware
- Integrate multiple industrial sensors
- Display live environmental data
- Record and validate sensor performance
- Implement wireless telemetry
- Build an industrial-quality engineering portfolio project
